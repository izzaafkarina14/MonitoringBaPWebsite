Profile BaP Monitoring System

Sistem Monitoring Konsentrasi Benzo[a]pyrene Menggunakan Sensor MQ-4 Berbasis Mikrokontroler Robotdyn Arduino Mega 2560 WiFi dengan Penampil Berupa Website

Pengembangan sistem monitoring berbasis sensor MQ-4. Menurut datasheet sensor MQ-4, biasa melakukan pendeteksian pada gas seperti CH4, LPG, dan CNC. Namun, pada penelitian ini sensor tersebut digunakan untuk melakukan pendeteksian konsentrasi senyawa benzo(a)pyrene pada beberapa sumber dalam pembakaran. Tahap pengembangan pada sistem ini menggunakan komunikasi nirkabel sebagai bentuk pengiriman data dengan media berupa real time database Firebase yang akan ditampilkan pada website monitoring dengan Google Spreadsheet sebagai penyimpanan data hasil monitoring karena tidak memiliki keterbatasan jumlah data. Tujuan dilakukan pengembangan tersebut yaitu agar mampu melakukan monitoring kualitas udara hasil pembakaran beberapa sumber secara real time. Prosesor yang digunakan yaitu menggunakan mikrokontroler Robotdyn Arduino Mega 2560 WiFi R3.
